    • AC Proqramlama dili Icersinde ferqli proqramlama dilerini birlesdirereq ortaya yeni ve daha ferqli bir proqramlama yapisi ortaya qoyur.


  • AC Proqramlama Hnsi Diler Iyle Yazilib

    • AC Proqramlama Dili Java iyle yazilib ve ardindan C iyle bezi hisselerde

      desteklenib gelecekde Python proqramlama dilinden istifade edilmesi gozlenilir.

      Qubik-ler


  • Qubik-ler Nedir?

    • Qubik-ler AC Proqramlama ile Proqramsitliqe daxil olunan bir tanimdir bu

      tanim sayesinde siz AC Prqoramlama dilinde tanimlanan ozel proqramlama dilerini daxil ederek o dilerin oxunmasi ve AC terefinden tehliledilmesi ni heyata kecire bilersiniz.


  • Qubik-ler Nece Yazilir?

    • Qubik-ler Ozel olaraq bu karakterler terefinden ac proqramlama ya

      tanimlandirilir < > ve bunlar icersinde ozel olaraq tanimlanan bezi kod lar daixl olunaraq istenilen hedefe catilir meseln: <OZEL KOD: IKINCI OZEL KOD; ICERIK SON AC> daha aydin anlamaniz uycun asaqida yazilan verilen kodlara baxa bilersiniz.


  • Ozel Olaraq Nece Qubik AC Terefinden Tanimlanib

    • <USTGLOBAL JAVA,C: deyisenler >

    • <BASLQNIC JAVA: javafayil.java; JAVA CODE SON AC>

    • <BASLQNIC C: cfayil.c; C CODE SON AC>

      istifade ede bilersiniz, ve bu sayede bir deyisgeni hem c den hem java dan hemde ac proqramlama dilerinden deyisib tekrar geri oxuma yerine yerine yetire bilersiniz.


  • Nece Istifade Olunur?

    • Mesela: <USTGLOBAL JAVA,C: herif ac = “Salam AC”; reqem ac = 3;

  • Icersinde Deyisen Tanimlama Acar Sozleri

  • EDCON Nədir?

    • EDCON Ozel olarq AC-T-İDE icersinde var olan bir ozelikdir bu ozlik

      sayesinde siz kod editorunda kod yazdiqiniz zaman bezi ozel Acar sozler sayesinde isinizi rahatlasdira bilerisniz bu acar sozler sayiesinde kod fayilini saxlaya bilersiniz hansi fayil yolunda oldugunuzu oyrene bilerisniz vs olaraqda kod fayilini islede bilerisniz.


  • EDCON Problemler

    • EDCON İstiafde edeceyiniz kod fayilinda onceden EDCON-un Acar

      sozleriniz yazmamaqa diqet edin ki İsdemediyimiz bir problemle qrsi qrsiya gelmeyey. Qrasi gelinen Problemler

      • Problem Onceden Yazilmis Acar sozler EDCON aktif edildikden sonra

        ust hiselerde kod icersinde yazilan acar sozler tekrer oxunur ve bu iyse problem ortaya cixarir

      • Problem EDCON AC-T-İDE icersinde yer aldiqi uysun Kodunuz

        icersinde yanliz MODON acar sozunu istifade ede bilmeyeceksiniz cunku bu acar soz yalniz EDCON-un aktif etmek uycun var ve siz onu daixl ediyiniz zaman hemin acar soz silinerek EDCON komutu olaraq qebul edilir.

  • Java AC Fonksiyonlar Niye Var?

    • Bu fonksiyonlar sizin java iyle ac arasinda baglanti qurmaqiniza yardimci

      olacaq. AC kod icersinde tanimlanan deysienlerin java da istifade olunmasina komek eden fonksiyonlar ve istifadeciden melumat oxuyan ozel java fonksiyonlari da movcudur ki biz bunlardan cox tez-tez istifade edceyik bunlar sayesinde xetasiz bir istifadeci melumat oxuma islemi heyata kecireceyik.


  • Java AC Fonksiyon Xetalari

    • Bezi veziyetlerde ozel fonksiiyonlari istifade etmemek isinizi yarida qoya

      biler bunlardan bezilerini sizin uycun izah ve problem list olaraq asaqida qeyd edeceyem.

      • Problem Java uycun yazilan fonksiyonlari istiafde etmek uycun paket

        adlarini istifade etmek gerekir bu problemi en qisa zmanda hell etmek lazimdir.

      • Problem Windows icersinde Java kodunu islederken Melumat duzgun ve xetasiz bir sekilde Yazmaq uycun asqida qeyd edilen Yazdirac(String)

        Fonksiyonu istiafde etmek lazimdir bu fonksiyon sayesinde problemsiz bir sekilde isdediyinzi melumat ekrana yazdira bilersiniz. Onemli Not“Bu Ynliz Windows Emeliyat Systeminde Kecerlidir”

      • Problem Oxuma Islemi heyata kecirmek uycun Asaqidaki Oxuma Fonksiyonlarinda Istifade etmeyiniz lazimdir.

  • Yazdirac(String)

    • Bu fonksiyon widnows icersinde istifade olunmalidir bu fonksiyonla ekrana xetasiz melumat yazdirmaq mumkundur.

  • UG(String)

    • Bu fonksiyon sayesinde java icersinden AC USTGLOBAL Qubik-inde yer

      alan deyisenlere el catan olmasi uycun isdifade olunur bu fonksiyon sadece bir melumat alir bu melumat iyse USTGLOBALIN icersinde yer alan deyisenlerin adi olacaq ve geriye hemin deyisenin melumatini gonderecek.

  • UG(String, String, String)

    • Bu Fonskiyon sayesinde AC USTGLOBAL deyisgenini icerigini deyise

      bilerisniz veya basqa bir deyisgenin icerigini var olan deyisgein iocerigini pozmadan ona elave ede bilerisniz bunu istifade etmek uycun size iki operator verilir ama ilk once fonksiyonun istediyi uc String Melumat in ne olduquna nezer yetirek bunalrdan biri USTGLOBAL Deyısgen adi dir ikinci opeartordu operatorlar iyse bunlardir “ = +=” bunlardan ilki “=” var olan USTGLOBAL deyisgen icerigini vereceyiniz Deysigeinin icerigi eder veya melumatin, bu iyse “+=” varolan USTGLOBAL Deyisgeninin Deyismeden ve silmeden onun davami olaraq vereceyiniz deyisgen melumati veya icerig olaraq ardina elave eder Ucuncu String melumati iyse Bir String Deyisgeini veya bir String icerigi vermey lazimdir.


  • UGart(String, String, String)

    • Bu fonksiyon uc String Melumati alir bunlardan ilki USTGLOBAL

      icersinde yer alan deyisgenin adi ikinci String Melumati Operatordu bu Var olan USTGLOBAL deyisgenini hansi emeliyatla cagiracaqmizi bildirir var olan operatorlar “ +-*/ ” Bunlardan Biri Artirma Biri Azaltma Biri Vurma ve biri iyse Bolme Operatorlaridir bu operatorlardan biri yazdiqdan sonra ucuncu String Melumatini Daxil edirik o string melumati ya bir deyisgen ve ya da USTGLOBAL deyisgenimizin tipine uygun bir melumat olmalidir Onemli Not “Daxil edeyceyiniz Ucuncu String melumati bir String deyisgen icerigi veya bir String icerigi olmalidir”


  • Oxuyucu(String)

    • Bu fonksiyon sayesinde siz AC cod Icersinde Yazdiqiniz Java kodundan

      isdifadeciden melumat oxuma islemi heyata kecire bilerisiniz ve bu oxuduqunuzu size geri string olaraq geler.


  • OxuyucuInt(String)

    • Bu fonksiyon sayesinde siz AC cod Icersinde Yazdiqiniz Java kodundan

      isdifadeciden melumat oxuma islemi heyata kecire bilerisiniz ve bu oxuduqunuzu size geri int yeni Integer REQEM olaraq geler.

  • OxuyucuChar(String)

    • Bu fonksiyon sayesinde siz AC cod Icersinde Yazdiqiniz Java kodundan

      isdifadeciden melumat oxuma islemi heyata kecire bilerisiniz ve bu oxuduqunuzu size geri Char bir melumat olaraq geler.

  • printach(char sentence, int rt)

    • bu fonksiyon widnows da ekrana xetasiz bir sekilde melumat yazdirmaqmiza komey eder ve ekrana herif String tipinde melumat yazdirar.


  • printacr(char sentence[], int rt)

    • bu fonksiyonla windows ciersinde ekrana isdediyimiz yazini xetasiz bir sekilde yazdirmaq mumkundur bu fonksiyon ekrana reqem yeniki integer int yazdirir.


  • acOxuyucuInt(N)

    • Bu fonksiyonla C den AC icersinden isdifadeciden Integer yeni int bir deyer

      oxuya bilersiniz. Bu fonksiyonu istifade ediyiniz zaman istifadeci ekranda bir dialog ekrani gorecey ve bu ekrana melumatini daxil edcey siz isterseniz N yeni Fonksiyonun melumat alici qismine istifadecinin gormesini isdediyinzi melumati yazdira bilersiniz.


  • acOxuyucuString(N)

    • Bu fonksiyon iyle C den AC icersinden Isdifadeciden melumat oxuma yeni

      String Herif Melumati oxuya bilersiniz, bu fonksiyonu cagirdiqiniz zaman ekranda bir dialog mesaji cixacaq isdifadeci bu dialoq mesjina melumati daxil edecek ve siz fonksiyonun melumat alicisina N olaraq Isdifadecinin gormesini isdediyiniz melumati daxil ede bilersiniz.


  • UGReqem(Herif+Char)

    • Bu fonksiyon sayesinde C icersinden AC kod icersinde tanimlanan

      USTGLOBAL deyisenini almaq uycun istifade olunur, bu fonksiyonun melumat aclici qismi Herif yeni Char bir melumat alara bu iyse UTGLOBAL icersinde tanimlanan deyigenin adidir bu adi daxil ederek isdediyiniz USTGLOBAL deyisgeini ala bilersiniz. Ve bu fonksiyon sadece Integer yeni int reqem melumati alar yeni USTGLOBAL icersinde tanimladiqiniz deyisgeinin tipi reqem olmalidir.


  • UGHerif(Herif+Char)

    • Bu fonksiyon sayesinde C icersinden AC kod icersinde tanimlanan

      USTGLOBAL deyisenini almaq uycun istifade olunur, bu fonksiyonun melumat aclici qismi Herif yeni Char bir melumat alara bu iyse UTGLOBAL icersinde tanimlanan deyigenin adidir bu adi daxil ederek isdediyiniz USTGLOBAL deyisgeini ala bilersiniz. Ve bu fonksiyon sadece Integer yeni int reqem melumati alar yeni USTGLOBAL icersinde tanimladiqiniz deyisgeinin tipi herif olmalidir.


  • UGart(Herif+Char, Herif+Char, Reqem+Int)

    • bu fonksiyon sayesinde AC kod icersinde tanimlanan USTGLOBAL

      deyisgenlerini artira bilersiniz bu fonksiyonu istifade etmek uycun uc melumat daxil etmelisiniz bunlardan ilk USTGLOBAL deyisgen adi dir ikinci operatirlardir “+-*/” bunlardan birini daxil ederek ucuncu vereceyiniz

      Integer yeni int bir reqem melumati iyle Aritmetik yeni A.B.Y.V islemi Heyata kecire bilersiniz.


  • UG(Herif+Char, Herif+Char, Herif+Char,)

    • bu fonksiyon sayesinde isdediyiniz USTGLOBAL deyisgeninin iki ferqli

yontemle melumatini deyise bilersiniz bunu heyata kecirmek ucycun ilk olaraq USTGLOBAL deyisgenin melumatini temizleyib yerine yeni melumat daxil etmek ve bunu heyata kecirmek ucyun fonksiyona ilk olaraq birinci melumat Herif yeni char USTGLOBAL deyisgeinimiz adini daixl edirik ve ardindan Operator olaraq bu Operatoru daxil edirik “=” ve sonra iyse ucuncu olaraq melumatimizi daxil edirik bu bir herif yeni char bir melumat veya deyisgen olmalidir, daha sonra biz USTGLOBAL deysgenimzinin melumatini silmeden ve onu sifirlamadan var olan melumatin devami olaraq bu yola deyise bilerik operatir hisesine “+=” bu operatoru daxil edersek USTGLOBAL deysigenimizn melumatini silmeden onu deyismis olariq.
